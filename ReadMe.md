## Suggestions 

> if you want to start a similar project

1. 对于Qt的类和方法的疑问都按照以下优先级:

   [`Qt-Doc`](https://doc.qt.io) -> `StackOverFlow` -> `google` -> `国内网站, 如CSDN`

2. 强烈建议给浏览器安装`ChatGPT`插件(安装方法STFW), **很多**时候能够直接以代码的方式告诉你如何处理你并不了解的代码框架

3. 不要花太多时间在寻找图片/gif素材上, 虽然国内的素材也很少, 推荐一个我找素材的网站[爱给网_音效配乐_3D模型_视频素材_免费下载 (aigei.com)](https://www.aigei.com/), 也可以直接使用本仓库中的素材

4. 推荐一些实用的工具:

   1. [NickeManarin/ScreenToGif: 🎬 ScreenToGif allows you to record a selected area of your screen, edit and save it as a gif or video. (github.com)](https://github.com/NickeManarin/ScreenToGif) 非常使用的gif制作工具, 如果有需要制作或者翻转gif的需求, 这个软件很好用!
   2. github desktop 之前一直是手动bash来完成git操作, 知道安装了这个软件, 对git操作十分友好.
